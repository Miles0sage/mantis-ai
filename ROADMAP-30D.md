# Mantis — 30 Day Build Roadmap

## Goal

Ship Mantis as a real low-cost product, not just an agent framework.

## Week 1

- Background jobs with persistent status
- Session persistence in web UI
- Checkpoint / resume primitives

## Week 2

- Live task tree polish
- Exact model-aware cost accounting
- Hard budget limits that stop execution

## Week 3

- Approval UI for risky tools
- Multi-file diff preview
- Strong-model escalation rules for cross-file edits

## Week 4

- Resume interrupted jobs from checkpoints
- Demo flow and launch assets
- README repositioning around low-cost async execution

## Launch Message

Mantis should launch as:

**Open-source coding agent with background jobs, cost controls, and cheap-by-default model routing.**

Not:

**Another open-source Claude Code clone.**
